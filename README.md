# PerfTrack Athletes

**Информационная система для фиксирования показателей эффективности профессиональной деятельности спортсменов**

## Описание проекта

Веб-приложение для тренеров и спортсменов, позволяющее вести учёт спортивных результатов, отслеживать динамику показателей, управлять секциями и обмениваться объявлениями. Система разделена на две роли: **Тренер** и **Спортсмен**.

## Цель проекта

Разработка современной информационной системы для автоматизации процесса сбора, хранения и анализа показателей эффективности спортсменов с использованием современных веб-технологий и облачных сервисов.

## Основные возможности

### Для тренера:
- Регистрация и авторизация
- Создание и управление своими секциями
- Просмотр и одобрение заявок от спортсменов
- Добавление результатов спортсменам
- Публикация объявлений с привязкой к секциям
- Просмотр и редактирование профиля
- Удаление спортсменов из своего списка

### Для спортсмена:
- Регистрация и авторизация (включая Google)
- Отправка заявки на тренировки к тренеру
- Просмотр своих результатов с графиками и статистикой
- Просмотр объявлений от тренера
- Редактирование своего профиля (включая email и телефон)

## Технологии

- **Frontend**: Vite, HTML5, CSS3 (Tailwind CSS), JavaScript (ES6+)
- **Backend / База данных**: Firebase (Authentication, Realtime Database, Storage)
- **Визуализация**: Chart.js
- **Хостинг**: Firebase Hosting (опционально)

```

```

## Запуск проекта локально

```bash
# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev
```

Приложение будет доступно по адресу: `http://localhost:5173`

## Деплой на Firebase Hosting

```bash
npm run build
firebase deploy
```
